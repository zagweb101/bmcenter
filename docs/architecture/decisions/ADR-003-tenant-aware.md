# ADR-003: organization_id من اليوم الأول

- **الحالة:** Accepted (مثبّت في PRD 4.0 — ملحق ج)
- **التاريخ:** 2026-06-15

## القرار
كل كيان جذري يحمل `organization_id` (والمرتبط بالموقع `branch_id`) دون عزل كامل في MVP-0؛ الخادم يفرض Organization Scope ولا يثق بقيمة المؤسسة من الواجهة.

## السياق
مرجع: PRD 4.0 §12.
