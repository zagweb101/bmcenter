# ADR-001: توحيد تهجئة العلامة على baytalmoswer

- **الحالة:** Accepted (مثبّت في PRD 4.0 — ملحق ج)
- **التاريخ:** 2026-06-15

## القرار
تُستخدم `bayt-almoswer` في المستودعات والحزم والبيئة والنطاقات؛ الجذر المعتمد `baytalmoswer.net`. أي تهجئة أخرى تحويل 301 فقط ويُمنع وجود التهجئتين في مخرج إنتاجي.

## السياق
مرجع: PRD 4.0 §3.
